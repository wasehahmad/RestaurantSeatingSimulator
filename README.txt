README

1. Create a seating arrangement based on the one already preset into the program.
2. Remember that the table which seat fours and eights are array lists which have size 5 each i.e. they take an entire group into one slot which represents a table
3. The bar and round tables each have to have each member of the group added to consecutive seats.
4. Adjust the parameters defined as constants
5. Run the program which will print out a group number, the wait time to be seated and the estimated wait time in each separate line
6. At the end, the number of people who were added to the queue and the number of people who were seated is listed.
7. The number of groups of each size which were seated is also printed.
